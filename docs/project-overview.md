# Project Overview

## Project: reservation-management

### Executive Summary

This project is a web application for managing reservations. It caters to two types of clients: internal (government-affiliated) and external (non-government). Internal clients can reserve facilities for free with approval and documentation, while external clients can reserve for a price.

### Technology Stack

- **Framework**: Vue.js
- **Language**: TypeScript
- **State Management**: Pinia
- **Router**: Vue Router
- **UI**: Tailwind CSS, Swiper
- **Build Tool**: Vite
- **Testing**: Vitest, Playwright

### Architecture

The application follows a component-based architecture, which is typical for Vue.js projects. It is a monolith application.
