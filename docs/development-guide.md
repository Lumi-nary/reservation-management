# Development Guide

## Prerequisites

- Node.js: `^20.19.0 || >=22.12.0`

## Installation

```bash
npm install
```

## Development

To start the development server, run:

```bash
npm run dev
```

## Build

To build the project for production, run:

```bash
npm run build
```

## Testing

To run unit tests, run:

```bash
npm run test:unit
```

To run end-to-end tests, run:

```bash
npm run test:e2e
```
